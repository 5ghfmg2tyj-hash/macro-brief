    About this app

    Shared viewer for published macro briefs, allocation history, and money flow snapshots.

    How it works

    The published site is read-only for viewers. A trusted publisher account fetches market data, generates the brief with its own API key, and updates the site artifacts for everyone else to read.

    Data sources

    Bundled JSON for historical tilts and flows. Live quotes are fetched directly by the app from:

    • FRED CSV endpoints — Treasury yields, OAS spreads, WTI, gold fix, Fed funds.
    • Yahoo Finance chart API — equity indices and ETFs.
    • Stooq — fallback for daily closes.
    • CoinGecko — BTC spot.

    Flow data

    ETF and MMF flow views are now derived from the published daily flow history, with weekly points rolled up into trailing 7-day totals at each brief date.

    Source

    Code lives on GitHub. The recommended deployment path is Cloudflare Pages protected by Cloudflare Access.

    Settings

    Configure your AI provider and API key for brief generation. Keys are stored encrypted in the macOS Keychain.

    AI Provider

    API Key

    The key is encrypted with the macOS Keychain before being written to disk. It is never sent anywhere except the selected AI provider's API endpoint.

    Scheduled Brief Generation

    —

    When enabled, the app launches silently at login with no window. It generates the brief at the scheduled time and sends a notification — click it to open the app.
